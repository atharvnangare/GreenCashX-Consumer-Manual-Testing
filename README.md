# GreenCashX Consumer Manual Testing

This repository contains the manual testing deliverables for the **GreenCashX Consumer Module**. The testing was performed to validate the application's functionality, user interface, search capabilities, and data integrity.

## Testing Scope

The following testing activities were performed:

- Functional Testing
- UI Testing
- Search Testing
- Data / Value Integrity Testing
- Bug Reporting
- Test Execution Reporting

## Modules Tested

- Dashboard
- My Usage
- Wallet
- Marketplace
- My Orders
- Purchase History
- Settings
- KYC Status

## Outcome

The GreenCashX Consumer Module was tested for functionality, UI behavior, search operations, and data consistency. A total of **230** test cases were executed across Functional, Search, and Data Integrity testing. **13 functional defects** were identified and documented, while Search Testing and Data Integrity Testing completed successfully without any defects.
